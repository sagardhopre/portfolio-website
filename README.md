# Portfolio Website

This is a Flask-based portfolio website. Below are the instructions for setting up and deploying the application.

## Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone https://github.com/sagardhopre/portfolio-website.git
   cd portfolio-website
   ```

2. **Create a Virtual Environment**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the Application**
   ```bash
   flask run
   ```

## Deployment Instructions

To deploy the portfolio website, follow these steps:

1. **Choose a Hosting Platform** (e.g., Heroku, AWS, DigitalOcean)
2. **Set Up Environment Variables** (if required)
3. **Deploy the Application** using the chosen platform's guidelines.
   
### Note:
Make sure to configure any necessary settings for production, such as database connections and security settings.
